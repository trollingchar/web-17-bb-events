🎮 Battle Brothers Event Converter

Convert game event code → Fandom Wiki wikitext

📚 Instructions for Contributors

🔧 Getting Started

  • Decompile the game code first. See guide: Mods and Modding
  • Event files are in scripts/events/events/ (also check DLC folders)
  • Each event state = one Tabber tab. Find them in this.m.Screens.push({...})

📝 Converting Event Screens

  • ID = "A" → Use as Tabber tab name (|-|A=)
  • Text = "..." → Paste into left panel. Converter handles messy code; just avoid capturing extra quotes
  • Options = [...] → Button actions:
    • return "B" → Link to tab B
    • return 0 → Event ends (no link)
    • Add requirements/side effects to button text
  • function start → Character tokens (this.Characters.push) and effects happening when the screen is shown. There are bugs sometimes, add them to "Other notes" section
  • function onUpdateScore → Event conditions and variable setup. Cross-check with wiki (it's sometimes incorrect or outdated)

✍️ Formatting Guidelines

  • Use {{EventVariable|Name}} for character names (Spiderbro, Randombrother), edit something that makes no sense (Noncom) and trim full names when there are also short ones (Monkfull → Monk)
  • Remove backslashes: \' → ', \\n → line breaks, parser can do it
  • Put explanations inside {{Tooltip|term|explanation}}, provide details whereever possible
  • Use Title Case for game terms, include + signs where present (Bro gains +2-4 Max Fatigue)
  • For random injuries: use icon for most severe possible outcome
  • Round mood effects UP for icons: effect_mood_X.png where X = mood name
  • Use {{EventVariable}} in buttons too, not just description text

⚠️ Pay attention

  • Game files contain typos! Check for extra brackets, single-option {} blocks
  • Converter works without quotes too (pastes everything) — useful for raw snippets
  • Image sizes vary: asset_moral_reputation.png isn't square (specify both dimensions), event_12.png is oversized (add size parameter)
  • Censor offensive terms per Fandom Policy
  • Format existing screenshots into galleries
  • Remove __FORCETOC__ when editing event pages

📋 Wikitext Syntax Reference

Full Event Structure:

{{Event}} <tabber> |-|A= ...content... |-|B= ...content... </tabber> {{EventEnd}}

Event Screen (Tab):

|-|Name= {{EventTab|header=Title|left=Character1's token|right=Character2's token}} description text... {{EventButtons}} {{EventButton|...}} {{EventTabEnd}}

With Effects Table:

|-|Name= {{EventTab|header=Title}} description... {{EventSeparator}} {| class="event-table" style="margin:auto; max-width:50%" |- | [[File:icon.png|x25px]] || Effect description |- | [[File:effect mood good spirit.png|x25px]] || {{EventVariable|Bro}} gets a {{Tooltip|good mood|+0.5 (Saw something funny)}} |} {{EventButtons}} {{EventButton|...}} {{EventTabEnd}}

Button Syntax (alternating: text | link | linkText):

{{EventButton|I hope it was worth it. (the event ends)}} {{EventButton||#B|Get back to work!}} {{EventButton||#Monk|What do the old gods say, {{EventVariable|Monk}}?|{{Sp}}(requires Monk)}} {{EventButton|Let's free her. (|#Good|80% Good|,{{Sp}}|#Bad|20% Bad|)}}

Tip: Use {{Sp}} for explicit spaces where needed.
